# Contributing

Thanks for wanting to contribute to payme!

Pull requests, issues, and feature requests are all welcome and will be considered.

## Guidelines

1. Fork the repository and create your branch from `master`
2. Make your changes and test them locally
3. Submit a pull request with a clear description of your changes

## Commit Standards

This project follows [Conventional Commits](https://www.conventionalcommits.org/). Please format your commit messages accordingly:

```
feat: add new feature
fix: resolve bug
docs: update documentation
style: formatting changes
refactor: code restructuring
test: add or update tests
chore: maintenance tasks
```

## Code of Conduct

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).
